使用vscode编写python所需要的配置

最近在阅读ml-knn的多标记经典论文代码，同样又碰到了KNN算法，准备凑着集体智慧编程把这个知识点再过一遍，所以我竟然菜到还没有一份熟悉的编程软件。。。

所需要的前菜

VS code
VS code extension
Python3

一、VScode 与 extension的安装

  1. vscode的安装直接去官网下载就可以了

  2. 关于python extension需要在VScode里面的visual studio marketplace下载，这里搜索出来会有好多中选择，推荐的是published by Microsoft

    二、python interpret

    在安装完VScode和extension之后需要再安装它的interpret，这个直接去官网下载它的最新版本就好了

    安装结束后你需要验证一下是否安装成功了
    windows：open a command prompt

    py -3 –version
    如果输出的是你的python版本，那么就成功了

下载好python解释器，下一步就是告诉VScode它解释python所需要的解释器

opening the command Palette（ctrl+shift+p）
typing Python: Seclect interpret

以上如果提示安装成功，那么就算所需要的前菜就结束了

三、使用VScode 输出Hello World

在编写代码的过程中，如果需要自动补全的智能效果，需要安装这个Linting包
The Python extension also has full support for Linting.

四、python中的安装和使用包

例如你要运行下面这段程序

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 20, 100) # Create a list of evenly-spaced numbers over the range
plt.plot(x, np.sin(x)) # Plot the sine of each x point
plt.show() # Display the plot

可是运行的结果往往会变成这样

ModuleNotFoundError: No module named 'matplotlib'".

也就是我们缺少相应的matplotlib包，但是在python中为了避免不同包之间的冲突，我们需要避免把不同的包安装在一个全局的解释器环境当中，而是为不同的项目创造出对应的特有虚拟environment，步骤如下

1. Create and activate the virtual environment

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process

2. Select your new environment by using the Python: Select Interpreter command from the Command Palette.
3. Install the packages
4. Rerun the program now (with or without the debugger) and after a few moments a plot window appears with the output
